# Теоретическое введение
## Виртуальные машины

**Виртуальная машина (VM)** — программная среда, которая эмулирует аппаратное обеспечение компьютера и позволяет запускать операционные системы изолированно от основной системы.

## VirtualBox

**Oracle VM VirtualBox** — свободное программное обеспечение для виртуализации, позволяющее создавать и запускать виртуальные машины на различных хостовых ОС.

## Fedora Linux

**Fedora Linux** — свободный дистрибутив Linux, разрабатываемый сообществом и спонсируемый компанией Red Hat. Версия с оконным менеджером Sway обеспечивает легковесное окружение на базе Wayland.

# Контрольные вопросы
1. Какую информацию содержит учётная запись пользователя?
Учётная запись пользователя содержит:

Имя пользователя (username) — для входа в систему

UID (User ID) — уникальный идентификатор

GID (Group ID) — идентификатор основной группы

Домашний каталог (/home/username)

Командная оболочка (shell) — например, /bin/bash

Пароль (в зашифрованном виде в /etc/shadow)

Полное имя (GECOS) — опционально

2. Команды терминала
Для получения справки:

man <команда>        # руководство
<команда> --help     # краткая справка
Для перемещения по файловой системе:

cd <каталог>         # переход
cd ..                # на уровень выше
cd ~                 # в домашний каталог
pwd                  # текущий каталог
Для просмотра содержимого каталога:

ls                   # список файлов
ls -la               # подробный список со скрытыми
ls -lh               # с человеко-читаемыми размерами
Для определения объёма каталога:

du -sh <каталог>     # общий размер
df -h                # свободное место на дисках
Для создания/удаления каталогов/файлов:

touch файл           # создать файл
mkdir каталог        # создать каталог
rm файл              # удалить файл
rm -r каталог        # удалить каталог
cp файл1 файл2       # копировать
mv файл1 файл2       # переместить/переименовать
Для задания прав на файл/каталог:

chmod 755 файл       # права в цифровом виде
chmod u+x файл       # добавить выполнение для владельца
chown user:group файл # сменить владельца
Для просмотра истории команд:

history              # вся история
!!                   # повторить последнюю команду
!123                 # выполнить команду №123
Ctrl+R               # поиск по истории

3. Что такое файловая система?
Файловая система — способ организации и хранения данных на носителе. Примеры:

ext4 — стандартная для Linux, журналируемая

xfs — высокая производительность, большие файлы

4. Как посмотреть подмонтированные файловые системы?

mount                # список всех подмонтированных
df -hT               # с размерами и типом ФС
findmnt              # древовидный вывод
cat /proc/mounts     # информация из ядра

5. Как удалить зависший процесс?
bash
ps aux | grep процесс    # найти PID процесса
kill -9 <PID>            # принудительно завершить
pkill имя_процесса       # по имени
htop                     # интерактивный менеджер процессов
