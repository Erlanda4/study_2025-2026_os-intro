## Основные команды `tmux`

<div style="display: flex; gap: 20px;">

<div style="flex: 1;">

**Управление окнами**
- `Ctrl+b c` — создать новое окно
- `Ctrl+b w` — выбрать окно из списка
- `Ctrl+b 0-9` — переключиться по номеру
- `Ctrl+b ,` — переименовать текущее окно

**Управление панелями**
- `Ctrl+b %` — разделить окно по вертикали
- `Ctrl+b "` — разделить окно по горизонтали
- `Ctrl+b стрелка` — перемещение между панелями
- `Ctrl+b x` — закрыть текущую панель

</div>

<div style="flex: 1;">

**Другие полезные команды**
- `Ctrl+b d` — отключиться от сессии (detach)
- `Ctrl+b [` — войти в режим копирования
- `Ctrl+b ]` — вставить текст из буфера
- `Ctrl+b :` — открыть командную строку `tmux`
- `exit` — полностью завершить работу `tmux`

</div>

</div>

<div style="flex: 1;">

- Создание нескольких окон в одной сессии
- Разделение окна на независимые панели
- Быстрая навигация между панелями с помощью клавиш
